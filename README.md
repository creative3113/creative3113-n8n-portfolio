## My Workflows (Portfolio)

All JSON files are located in the [`workflows/`](./workflows/) folder. Each one can be imported into n8n instantly.

1. **Telegram AI Assistant**  
   Personal AI assistant inside Telegram: answers questions, performs web searches, sets reminders, and uses custom tools.  
   
2. **Personal AI Life Manager**  
   AI-powered life manager: daily summaries, habit tracking, task reminders, calendar & notes sync via Telegram. Includes long-term memory.  
  
3. **AI Customer Support Agent**  
   Smart support agent with RAG: uses your documents (PDFs, Google Docs, etc.), supports handover to human, works via Telegram or WhatsApp.  
  
4. **AI Social Media Manager**  
   Automated social media content engine: generates posts with AI, detects trends, schedules content for Instagram, Twitter/X, LinkedIn.  
 
5. **RAG Knowledge Base Agent**  
   Retrieval-Augmented Generation agent for knowledge bases: chunks documents, performs vector search, answers questions via Telegram or webhook.  

6. **AI Data Analyst**  
   Data analysis agent: ingests CSV/Sheets data, generates insights, reports, charts, and automatic SQL queries.

## How to Use
1. Open your n8n instance (local or cloud).
2. Go to **Import** → **From URL** or **From File**.
3. Paste the raw JSON link or upload the file directly.
4. Configure credentials (API keys for OpenAI, Groq, Pinecone, etc.).
5. Activate and test the workflow.
